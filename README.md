# dotfiles

Settings and configs for the tools I use

### Configs / Settings included:

- [x] NeoVim
  - Note: install [ripgrep](https://github.com/BurntSushi/ripgrep), [fd](https://github.com/sharkdp/fd) and [fzf](https://github.com/junegunn/fzf) for best experience with Telescope and Co.
- [x] LazyGit
- [x] Bash
- [x] Windows Terminal
- [x] Firefox

### Installation

There is no automated insallation process.

Where to put which config?
| Config | Linux | Windows |
| :----- | :------------- | :---------------------------- |
| NeoVim | ~/.config/nvim | %HomePath%\AppData\Local\nvim |
| Bash | ~ | %HomePath% |
| |
